由于时间关系，这里简述一下GitHub使用：
* 第一次使用GitHub，首先要到www.github.com注册一下账号。
* 下载安装<a href="https://windows.github.com/">GitHub客户端</a>。
* 安装完成之后，打开桌面上的GitHub图标，利用图形界面配置自己刚刚注册的Git帐号和密码。
* 然后从远程库克隆到本地。
